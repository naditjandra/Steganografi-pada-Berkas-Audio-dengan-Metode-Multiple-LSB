# Deskripsi
Perangkat lunak ini adalah sebuah aplikasi penerapan konsep kriptografi dan steganografi yang berusaha mengenkripsi dan menyisipkan sebuah file berbasis bit ke dalam sebuah file audio. Program ini memungkinkan pengguna untuk menyisipkan dan mengekstrak file dalam format audio MP3/WAV dengan aman tanpa menyebabkan perubahan signifikan pada kualitas suara. Untuk meningkatkan keamanan, program ini juga mendukung enkripsi file menggunakan Extended Vigen√®re Cipher, yang beroperasi pada data 256-byte. Melalui kombinasi konversi berkas dan manipulasi tingkat bit, program ini memastikan penyembunyian dan pengambilan data yang andal yang secara efektif menunjukkan bagaimana steganografi dan kriptografi dapat bekerja sama untuk melindungi informasi sensitif. <br><br>

# Tech Stack
Programming Language : Python Version 3.12.10 <br>
Python Modules : os, wave, struct, pydub <br>
Input Audio : Any audio files ('.wav', '.mp3', '.flac', '.ogg', '.aac', '.m4a') <br>
Secret Files : Any file type (binary based) <br><br>

# Dependencies (This is only relevant for using the source code)
Programming Language : Python Version 3.12.10 or lower (Version 3.13 or higher wouldn't work because of dependencies on audioop library which is removed on version 3.13) <br>
Python Modules : os, wave, struct, pydub <br><br>
